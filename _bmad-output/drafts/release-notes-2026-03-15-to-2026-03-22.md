## v2.4.0 — Enhance Module (March 15)

The headline release of the week. Convoke now has two extensibility axes: **Teams** (new agents) and **Skills** (new capabilities for existing agents).

### Added

- **Enhance module — Agent Skills architecture** — New module type alongside Teams. Skills add capabilities to existing agents via menu patching without modifying agent files
- **Initiatives Backlog skill (PM agent)** — RICE-scored backlog management with 3 modes:
  - **Triage** — Ingest review findings, extract actionable items, two-gate RICE scoring, append to backlog (4 step files)
  - **Review** — Walk through items, rescore where priorities shifted, regenerate prioritized view (3 step files)
  - **Create** — Gather initiatives interactively, batch-score, generate complete backlog from scratch (4 step files)
- **Installer integration** — File copy, PM agent menu patching (`[IB] Initiatives Backlog`), 6-point `convoke-doctor` verification, skill wrapper and manifest registration
- **ENHANCE-GUIDE.md** — Pattern documentation for module authors: directory structure, step files, menu patching, config, verification
- **Sprint retrospectives** — Retros captured for all 3 Enhance epics (Epic 1: infrastructure, Epic 2: review mode, Epic 3: create mode + guide)

### Changed

- **README** — Repositioned around Teams + Skills. Full Enhance section with mode diagram, activation instructions, and pattern guide link
