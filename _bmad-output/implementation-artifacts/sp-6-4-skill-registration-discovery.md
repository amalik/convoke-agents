# Story SP-6.4: Skill Registration & Discovery

Status: done

## Story

As a user,
I want the new portability skills to appear in the help menu and be discoverable,
so that I know they exist without reading documentation.

## Acceptance Criteria

1. All 4 portability skills registered in `skill-manifest.csv` with correct tier, intent, and paths.
2. All 4 skills have `.claude/skills/<name>/SKILL.md` with frontmatter including "Use when..." trigger phrases.
3. Each skill's description is clear enough to identify its purpose from a list.
4. The `_bmad/bme/_portability/` module directory exists with all skill copies.
